Trackmania Colour Utility

I'll get some nice styling on this at some point but I wanted to quickly explain the purpose of this utility, as well as how to use it.

Purpose

The Trackmania Colour Utility provides an easy-to-use, simple way of filling any and all map signs with a solid colour for any possible hexadecimal code available. This should be a quicker, easier way of doing it than, say, making the image in Photoshop, uploading the image somewhere and then linking to that image from within the game's map editor.

Instead, you can now just paste in a link similar to the ones below, and instantly texture your signs with whatever colour you want!

How to use

This utility works by reading a hexadecimal code that you pass in via the URL (the text at the top of your browser). I think some examples might work better than me trying to explain it with words:

  • Red: https://trackmania-colour.herokuapp.com/ff0000.png
  • Green: https://trackmania-colour.herokuapp.com/00ff00.png
  • Blue: https://trackmania-colour.herokuapp.com/0000ff.png

Each of those links should take you to a page displaying a 1x1 pixel image, in the colour specified by the hexcode (those six letters/numbers) just before the .png in the URL.
